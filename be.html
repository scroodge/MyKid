Фота, відэа, моманты — захоўваюцца ў вашым Immich. Дзённік сінхранізуецца праз Supabase.

Скрыншоты


Магчымасці

  • Уласны хостинг — фота ў вашым Immich, даныя ў вашым Supabase
  • Сямейны доступ — запрашайце членаў сям'і, дзяліцеся дзецьмі і запісамі
  • Магія EXIF — дата і месца аўтаматычна падстаўляюцца з метаданых фота
  • AI-апісанні — генерацыя тэксту дзённіка з фота (OpenAI, Gemini, Claude або DeepSeek); гл. docs/ai-providers.md
  • Афлайн-рэжым — кэшаваны спіс запісаў пры адсутнасці сеткі
  • Масавы імпорт — даданне мноства фота за раз

Хуткі старт

flutter pub get
flutter run --dart-define-from-file=.env

Гл. Налада для канфігурацыі Supabase і Immich.

Налада

Патрабаванні

  • Flutter SDK (stable)
  • Android Studio / Xcode
  • Праект Supabase
  • Уласны Immich або публічны сервер

Пераменныя асяроддзя

Стварыце .env. Не каміцьце рэальныя ключы.

  • SUPABASE_URL — URL вашага праекта Supabase
  • SUPABASE_ANON_KEY — ананімны ключ Supabase

Backend (Supabase)

Стварыце праект на supabase.com, прымяніце схему docs/full_schema.sql, уключыце Email у Authentication.

Immich

У дадатку: Налады → Immich: пакажыце URL сервера і API-ключ.

AI-апісанні (опцыянальна)

Налады → AI-правайдеры — дадайце API-ключы і выберыце правайдера па змаўчанні. Падрабязней у docs/ai-providers.md.

Удзельнікі

Дзякуй усім, хто ўносіць уклад у MyKid. Гл. CONTRIBUTING.md.
