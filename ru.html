MyKid

Дневник жизни вашего ребёнка

Фото, видео, моменты — хранятся в вашем Immich. Дневник синхронизируется через Supabase.

Скриншоты


Возможности

  • Собственный хостинг — фото в вашем Immich, данные в вашем Supabase
  • Семейный доступ — приглашайте членов семьи, делитесь детьми и записями
  • Магия EXIF — дата и место автоматически подставляются из метаданных фото
  • AI-описания — генерация текста дневника из фото (OpenAI, Gemini, Claude или DeepSeek); см. docs/ai-providers.md
  • Офлайн-режим — кэшированный список записей при отсутствии сети
  • Массовый импорт — добавление множества фото за раз

Быстрый старт

flutter pub get
flutter run --dart-define-from-file=.env

См. Настройка для конфигурации Supabase и Immich.

Настройка

Требования

  • Flutter SDK (stable)
  • Android Studio / Xcode
  • Проект Supabase
  • Собственный Immich или публичный сервер

Переменные окружения

Создайте .env. Не коммитьте реальные ключи.

  • SUPABASE_URL — URL вашего проекта Supabase
  • SUPABASE_ANON_KEY — анонимный ключ Supabase

Backend (Supabase)

Создайте проект на supabase.com, примените схему docs/full_schema.sql, включите Email в Authentication.

Immich

В приложении: Настройки → Immich: укажите URL сервера и API-ключ.

AI-описания (опционально)

Настройки → AI-провайдеры — добавьте API-ключи и выберите провайдера по умолчанию. Подробнее в docs/ai-providers.md.
